# SixSeven Emoji Replacer

[Spicetify](https://github.com/khanhas/spicetify-cli) extension that replaces standalone `67` text with an animated glowing cyan inline emoji effect

* Simply run the extension and it will work!

## Install

Copy `sixseven.js` into your [Spicetify](https://github.com/khanhas/spicetify-cli) extensions directory:

| **Platform**   | **Path**                                                                             |
|----------------|--------------------------------------------------------------------------------------|
| **Linux**      | `~/.config/spicetify/Extensions` or `$XDG_CONFIG_HOME/.config/spicetify/Extensions/` |
| **MacOS**      | `~/spicetify_data/Extensions` or `$SPICETIFY_CONFIG/Extensions`                      |
| **Windows**    | `%appdata%\spicetify\Extensions\`                                                    |

After putting the extension file into the correct folder, run the following command to install the extension or install through Marketplace:

```sh
spicetify config extensions sixseven.js
spicetify apply
